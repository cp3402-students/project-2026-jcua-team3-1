# KC Tennis Blast – Custom WordPress Theme

## Overview

This repository contains the **KC Tennis Blast** custom WordPress theme, developed as part of the CP3402 project.

The theme is built on the Underscores (`_s`) starter framework and is designed to provide full control over layout, performance, and content structure. It intentionally avoids page builders and hardcoded content, making it lightweight, maintainable, and reusable across different WordPress installations.

For a detailed breakdown of how the theme is structured and implemented, see the [Theme Documentation](./theme.md).

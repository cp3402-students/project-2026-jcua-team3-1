# KC Tennis Blast Website

## Brief project overview

This repository contains a custom WordPress website built for a client, focused on delivering a clean, maintainable, and production-ready solution.

The site is developed using a custom theme based on a starter framework, allowing full control over design, performance, and content structure. It avoids page builders and hardcoded content, making it flexible and reusable across different WordPress setups.
